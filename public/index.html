Задача по js

Разработать всплывающее окно.

    Оформление:
  1. Окно с информацией должно быть закреплено в нижней части экрана
  2. Х для закрытия
    Скрипт:
  1. Закрывает на Х
  2. Срабатывает через 3 секунды после открытия страницы
  3. Если пользователь закрыл окно, при повторном открытии страницы не срабатывает
  4. Повторный вызов сработает через 3 минуты
  5. После сброса кеша браузера через 3 секунды после открытия страницы
