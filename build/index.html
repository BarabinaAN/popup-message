Разработать всплывающее окно:

    Оформление:
  1. Окно с информацией должно быть закреплено в нижней части экрана
  2. Х для закрытия
    Скрипт:
  1. Закрывать на Х или кликом вне окна
  2. Срабатывать через xxx секунд после открытия страницы
  3. Если пользователь закрыл окно, при повторном открытии страницы скрипт не срабатывает
  4. Повторный вызов скрипта сработает через xxx дней
  5. После сброса кеша браузера через xxx секунд после открытия страницы
